  SMARTGPU2 ARDUINO 1.0 LIBRARY V3.0

This library help the user to communicate the SMARTGPU2 BOARD with the arduino board, be sure to check pinout while mounting the SMARTGPU2!


Where to place this entire folder(SMARTGPU2)?

place it in the arduino installation folder path:

\arduino-1.0.X\libraries\SMARTGPU2

restart arduino IDE to recognize the library and enjoy!

be sure to check out all the SMARTGPU2 examples 100% ready to compile and load in our website
