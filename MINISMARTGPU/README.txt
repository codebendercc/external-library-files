  MINI SMARTGPU ARDUINO 1.0 LIBRARY

This library help the user to communicate the MINI SMARTGPU BOARD with the arduino board, be sure to check pinout while connecting the MINI SMARTGPU!

Where to place this entire folder(MINISMARTGPU)?

place it in the arduino PC root:

arduino-00XX\arduino-00XX\libraries\MINISMARTGPU

restart arduino to recognize the library and enjoy!

be sure to check out all the MINI SMARTGPU examples 100% ready to compile and load in our website
